# phaser-ts-gulp-sandbox

Projeto Sandbox para testes do Framework Phaser

## Executar o projeto

```npm install```  
```./node_modules/grunt-cli/bin/grunt watch```  

Em outro terminal:  
```./node_modules/grunt-cli/bin/grunt serve```
